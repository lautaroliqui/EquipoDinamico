Propósito del team

Este proyecto se centró en desarrollar una aplicación web para la administración de una biblioteca, combinando las bases de HTML, CSS, el framework Django y Bootstrap para el diseño. El principal objetivo fue crear una plataforma web funcional que permita gestionar los préstamos y la disponibilidad de libros en una biblioteca de manera eficiente y fácil de usar.

Este trabajo fue realizado como parte de un proyecto académico en la facultad, involucrando a tres integrantes. La finalidad era aplicar y consolidar conocimientos adquiridos en el curso, abarcando tanto el frontend como el backend del desarrollo web.

Este proyecto logró integrar y aplicar de manera efectiva tecnologías frontend y backend, resultando en una aplicación web bien diseñada para la administración de una biblioteca. Fue una excelente oportunidad para poner en práctica habilidades técnicas y de colaboración en equipo.
